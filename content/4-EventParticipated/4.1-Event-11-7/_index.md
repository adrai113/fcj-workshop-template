# AWS Community Meetup Report

## Event Information

- **Event Name:** AWS Community Meetup
- **Date:** July 11, 2026

## Event Introduction

This event is a meetup to share knowledge around web application security, system monitoring (SLA & Monitoring), and study roadmaps to conquer the AWS Cloud Practitioner certification. The speakers are experts and active members of the AWS community.

---

## Session Summaries

### 1. Securing Your Web Apps With AWS Security Agent

**Speaker:** Thinh Nguyen  
**From:** DevOps/DevSecOps/Cloud Engineer at Styl Solutions

This session focused on AWS Security Agent (Frontier Agent) powered by Amazon Bedrock, which automates security processes. Key features include:
- **Design Review:** Analyzes architecture documents before coding to check against security standards (PCI DSS, NIST CSF, AWS Well-Architected).
- **Code Security Review:** Automatically scans Pull Requests on GitHub/GitLab for vulnerabilities and suggests code patches (Auto-PR Fixes).
- **Automated Pentesting:** Performs automated penetration testing and validates findings like a real user.

---

### 2. SLA and Monitoring - From SLA to Monitoring what really matters

**Speaker:** Nguyễn Huỳnh Sơn  
**From:** AWS Student Builder Group HUFLIT Member

The presentation emphasized that "Healthy Infrastructure does not equal Happy Users." Instead of just looking at server metrics, we need to focus on business metrics and Customer Journey (like Login, Checkout).
- Introduced the risk management cycle: Identify -> Monitor -> Respond -> Improve.
- Showed practical examples of setting up CloudWatch Alarms and SNS Topic fan-out to alert immediately when metrics (e.g., Login Failure) cross thresholds.

---

### 3. Inside The Exam: AWS Cloud Practitioner

**Speaker:** Ngo Le Tan Huy  

The speaker provided a strategic roadmap to conquer the AWS Cloud Practitioner (CLF-C02) exam, including:
- **Exam Info:** 65 multiple-choice questions in 90 minutes, covering 4 domains (Cloud Concepts, Security and Compliance, Cloud Technology and Services, Billing).
- **Study Methods:** Map Keyword Thinking, analyzing mistakes from mock tests (Review Mistakes), and hands-on practice on AWS Free Tier.
- **Tips & Tricks:** Use elimination techniques, don't overthink, watch out for language pitfalls, and use the flag feature for hard questions.

---

## Key Takeaways

- Gained a better understanding of automated security tools and the role of AI in DevSecOps with AWS Security Agent.
- Developed a better mindset for Monitoring: always prioritize user experience when setting up monitoring and SLAs.
- Grasped the structure, review tactics, and useful tips to prepare well for the AWS Cloud Practitioner certification.
